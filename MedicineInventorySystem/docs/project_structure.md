# Project Structure

- README.md: project overview and setup steps.
- report.docx: report template content.
- presentation.pptx: slide content outline.
- video_link.txt: placeholder for demo video link.

- sql/: all database scripts.
  - create_tables.sql: table creation with keys.
  - load_data.sql: sample data for each table.
  - queries.sql: required query examples.
  - triggers.sql: billing and stock triggers.
  - views.sql: simple read-only views.

- src/: Flask application code.
  - app.py: main routes and logic.
  - database.py: MySQL connection helper.
  - templates/: HTML pages.
  - static/: CSS and JavaScript files.
  - requirements.txt: Python dependencies.

- docs/: documentation and diagrams.
  - ERD.png: ER diagram image placeholder.
  - schema.png: schema image placeholder.
  - database_explanation.md: table and relationship details.
  - sql_explanation.md: SQL scripts explanation.
  - flask_explanation.md: app flow and routes.
  - erd_explanation.md: ERD explanation.
  - setup_guide.md: install and run steps.
